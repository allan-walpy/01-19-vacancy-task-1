# 01-19-vacancy-task-1

## task

Make system for:

- forming vacation list;
- searching in list by parameters;

Vacancy consists of:

- title (vacancy name);
- salary;
- employer (as organization);
- contact person;
- contact phone;
- employment type (full-time etc.);
- description;
